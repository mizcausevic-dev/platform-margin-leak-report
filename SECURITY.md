# Security Policy

`platform-margin-leak-report` includes an offline analyzer, CLI, and public synthetic executive dashboard for platform margin posture. It does **not** store live billing credentials, perform live vendor fetches, or expose authenticated write paths.

## Reporting

- [Open a security advisory](https://github.com/mizcausevic-dev/platform-margin-leak-report/security/advisories/new)
- or open a private issue if advisory flow is unavailable

## Scope notes

- Synthetic sample data only
- Read-only routes and static prerender outputs
- No tenant credentials or finance-system secrets in repo
